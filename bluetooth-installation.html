8.2 Bluetooth Installation and Wiring

Development kit hardware specification

The Bluetooth development kit comes with the following:-
  • Development board: The motherboard is used to mount the chosen wireless module, providing RS232 interface and access to all the module interface pins.
  • USB cable: USB cable to provide power for development kit via onboard connector.
  • Bluetooth USB connector: Bluetooth module support Windows protocols stack.
  • Serial cable: RS232 interface cable to provide the communication between PC and wireless module.


The development board can be powered by:-
  • 5V power from USB port
  • External power supply via screw terminal
  • External main power block via 2.1mm DC connector
