9.3.2 Configure ZigBee RF module to Unicast communication mode

Allow embedded UNO only communicate with the dedicated I/O device without interrupting other Zigbee RF within the same network ID.


Understand the application program flow:-
6) Attach Zigbee RF module to embedded PC.
7) Create the program that may open Serial Port to communicate with the ZigbeeRF module.
8) Updating the Zigbee RF module parameters, configure the Zigbee RF modules which match with the Application scenario.
9) The parameters are:-
  • SL
  • SH
  • DL
  • DH
  • NETWORK ID
  • ROUTER
10) Repeat the Step (1) to Step (4) for other Zigbee modules which plan to be used in same network.
