6.3 GPS Installation and Wiring

Hardware Specification

GPS-102 is a GPS receiver module with integrated dead reckoning. The GPS-102 includes dead reckoning sensors to track its course even when the GPS signal is blocked, for example in urban areas or tunnels. If GPS coverage is lost, the GPS-102 keeps on navigating.



Software Features
- Detect GPS Signal Strength
- Detect Satellites
- Parse NMEA Protocols
